# 🎵 Deep Aura

A modern, sleek, and feature-rich music streaming web application. Deep Aura is designed to offer a seamless audio experience with dynamic track suggestions, integrated lyrics, and a highly responsive user interface. 

## ✨ Features

* **Dynamic Homepage:** Automatically updates with sections like 'Made For You' and 'Trending Now'.
* **Smooth Audio Playback:** Reliable backend streaming routes for uninterrupted music listening.
* **Integrated Lyrics:** View synchronized lyrics for your favorite tracks while they play.
* **Progressive Web App (PWA):** Fully responsive design that can easily be converted into a standalone Android APK.
* **Cinematic UI:** A visually stunning interface designed for maximum user engagement.

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js (Hosted on Replit)
* **Media:** HTML5 Audio API & Custom Streaming Routes
